# User Quickstart

Accelerate Global Data lets signed-in users browse shared people group datasets,
filter rows, and save useful table views.

## Access

- Use the sign-in page at `/`.
- New accounts must be on the signup allowlist before registration.
- Ask an administrator for access if signup is blocked.
- Password reset starts from `/forgot-password`.

## Browse Datasets

1. Sign in.
2. Open the dashboard.
3. Choose a dataset from the dataset list.
4. Use table controls to filter, sort, and inspect rows.
5. Save a filtered table when you want to return to the same view later.

## Saved Tables

Saved tables preserve a named filtered view of a dataset. They are useful for
repeat review, focused collaboration, and exporting a subset of rows.

## Profile

Use the profile page to review account details and sign out. If an account is
disabled, the user cannot sign back in until an administrator restores access.

## Admin Capabilities

Admins can manage datasets, upload or replace CSV data, edit field definitions,
review field source mappings, manage API connections, view analytics, and manage
users. Admin access is assigned outside the user-facing UI through the
environment/provider admin process.

## Support Notes

If expected data is missing or a page redirects unexpectedly, confirm that the
account is signed in and has the required admin role for the action.
